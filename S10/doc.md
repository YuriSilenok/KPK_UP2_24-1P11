# Employee Status Service

# EmployeeStatus

## 1. Добавить сущность

### Информация для создания

| Параметр (англ.) | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|---|
| employee_id | Идентификатор сотрудника из внешнего сервиса | Да | integer | > 0 | - |
| position_id | Идентификатор должности | Да | integer | > 0 | - |
| rate | Размер ставки | Да | float | > 0 | - |
| is_part_time | Признак совместительства | Нет | boolean | - | false |
| start_date | Дата начала действия статуса | Да | date | - | - |
| end_date | Дата окончания действия статуса | Нет | date | - | null |

### Уникальные комбинации параметров

Отсутствуют.

### Информация при успешном создании

| Параметр (англ.) | Тип |
|---|---|
| id | integer |

## 2. Изменить сущность по ID

### Информация для изменения

| Параметр (англ.) | Пояснение | Обязательность | Тип | Ограничение |
|---|---|---|---|---|
| position_id | Идентификатор должности | Нет | integer | > 0 |
| rate | Размер ставки | Нет | float | > 0 |
| is_part_time | Признак совместительства | Нет | boolean | - |
| start_date | Дата начала действия статуса | Нет | date | - |
| end_date | Дата окончания действия статуса | Нет | date | - |

### Информация при успешном изменении

| Параметр (англ.) | Тип |
|---|---|
| result | boolean |

## 3. Удалить сущность по ID

Удаление логическое.

При удалении поле `is_active` устанавливается в `False`.

Возвращаемое значение:

- true — запись найдена и помечена удалённой;
- false — запись не найдена.

## 4. Получить сущность по ID

### Возвращаемая информация

| Параметр (англ.) | Пояснение | Тип |
|---|---|---|
| id | Идентификатор записи | integer |
| employee_id | Идентификатор сотрудника | integer |
| position_id | Идентификатор должности | integer |
| rate | Размер ставки | float |
| is_part_time | Совместительство | boolean |
| start_date | Дата начала | date |
| end_date | Дата окончания | date |
| is_active | Активность записи | boolean |

## 5. Получить список сущностей по заданным параметрам

### Параметры запроса

| is_active | Активность записи | boolean |

### Возвращаемый список

|---|---|
| id | integer |
| employee_id | integer |
| position_id | integer |
| rate | float |
| is_part_time | boolean |
| start_date | date |
| end_date | date |
