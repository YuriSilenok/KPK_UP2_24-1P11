# Вариант №10. Сервис статуса сотрудника (Employee Status Service)

## Справочник должностей (Position)

### Создать должность (Create Position)

| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
| :--- | :--- | :--- | :--- | :--- | :--- |
| title | Название должности | Да | String | Макс. 255 символов | - |
| code | Код должности | Да | String | Уникальный, макс. 50 символов | - |
| department | Подразделение | Да | String | Макс. 100 символов | - |

**Уникальные комбинации параметров:**
*   Поле `code` должно быть уникальным в рамках всей системы.

### Получить список должностей (List Positions)

| Параметр фильтрации | Пояснение | Тип | Пример значения |
| :--- | :--- | :--- | :--- |
| department | Фильтр по подразделению | String | "IT" |
| is_active | Статус активности | Boolean | true |

### Обновить должность (Update Position)

**Ответ при успешном обновлении (200 OK)**

| Параметр | Пояснение | Тип |
| :--- | :--- | :--- |
| id | Идентификатор записи | Integer |
| title | Название должности | String |
| code | Код должности | String |
| department | Подразделение | String |
| is_active | Статус активности | Boolean |
| created_at | Дата создания | DateTime |
| updated_at | Дата последнего обновления | DateTime |

## Ставка сотрудника (EmployeePosition)

### Создать ставку (Create EmployeePosition)

| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
| :--- | :--- | :--- | :--- | :--- | :--- |
| profile_id | ID профиля сотрудника | Да | Integer | - | - |
| position_id | ID должности | Да | Integer | - | - |
| start_date | Дата начала | Да | Date | Не в будущем | - |
| end_date | Дата окончания | Нет | Date | После start_date | NULL |
| rate | Ставка | Да | Decimal | [0.1; 2.0] | - |
| is_primary | Основная ставка | Нет | Boolean | - | false |

**Уникальные комбинации параметров:**
*   У одного сотрудника может быть только одна активная основная ставка (`is_primary=true`).

### Получить список ставок (List EmployeePositions)

| Параметр фильтрации | Пояснение | Тип | Пример значения |
| :--- | :--- | :--- | :--- |
| profile_id | ID сотрудника | Integer | 101 |
| position_id | ID должности | Integer | 5 |
| is_active | Статус активности | Boolean | true |

### Обновить ставку (Update EmployeePosition)

**Ответ при успешном обновлении (200 OK)**

| Параметр | Пояснение | Тип |
| :--- | :--- | :--- |
| id | Идентификатор записи | Integer |
| profile_id | ID профиля сотрудника | Integer |
| position_id | ID должности | Integer |
| start_date | Дата начала | Date |
| end_date | Дата окончания | Date |
| rate | Ставка | Decimal |
| is_primary | Основная ставка | Boolean |
| is_active | Статус активности | Boolean |

## Отпуск (Leave)

### Создать отпуск (Create Leave)

| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
| :--- | :--- | :--- | :--- | :--- | :--- |
| employee_position_id | ID ставки сотрудника | Да | Integer | - | - |
| start_date | Дата начала | Да | Date | - | - |
| end_date | Дата окончания | Да | Date | После start_date | - |
| type | Тип отпуска | Да | String | Enum | - |
| status | Статус | Да | String | Enum | "PLANNED" |

**Уникальные комбинации параметров:**
*   Даты отпуска не должны пересекаться с другими активными отпусками или больничными на той же ставке.

| type | Тип отпуска | String | "ANNUAL" |

## Больничный (SickLeave)

### Создать больничный (Create SickLeave)

| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
| :--- | :--- | :--- | :--- | :--- | :--- |
| employee_position_id | ID ставки сотрудника | Да | Integer | - | - |
| start_date | Дата начала | Да | Date | - | - |
| end_date | Дата окончания | Да | Date | >= start_date | - |
| certificate_number | Номер листка нетрудоспособности | Да | String | Макс. 50 символов | - |
| status | Статус | Да | String | Enum | "OPEN" |

**Уникальные комбинации параметров:**
*   Даты больничного не должны пересекаться с другими активными больничными или отпусками на той же ставке.

| status | Статус больничного | String | "CLOSED" |
