# Вариант №11. Сервис дисциплин (Discipline Service)

## 1. Добавить дисциплину

Информация, требуемая для создания дисциплины:

| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|
| name | Обязательно | Строка | не пустое, уникальное | — |
| code | Обязательно | Строка | формат: буквы, цифры, точки, дефисы, уникальный | — |
| speciality_ids | Обязательно | Массив целых | Каждый ID должен существовать в таблице Speciality, минимум один элемент | — |

Параметры name и code уникальны глобально (не зависят от специальности). Дисциплина может быть привязана к одной или нескольким специальностям через speciality_ids.

Информация, возвращаемая в случае удачного создания дисциплины:

| Параметр | Тип |
|---|---|
| id | Целое |
| name | Строка |
| code | Строка |
| specialities | Массив объектов (каждый объект содержит id, name, code) |

## 2. Изменить дисциплину по ID

Информация, требуемая для изменения дисциплины по ID:

| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|
| name | Не обязательно | Строка | Уникальное, не пустое | — |
| code | Не обязательно | Строка | Уникальное, формат: буквы, цифры, точки, дефисы | — |
| speciality_ids | Не обязательно | Массив целых | Каждый ID должен существовать в таблице Speciality | — |

При изменении speciality_ids старые связи заменяются новыми (сначала удаляются все существующие связи дисциплины со специальностями, затем добавляются новые).

Информация, возвращаемая в случае удачного изменения дисциплины:

| Параметр | Тип |
|---|---|
| id | Целое |
| name | Строка |
| code | Строка |
| specialities | Массив объектов (каждый объект содержит id, name, code) |

## 3. Удалить дисциплину по ID

Вернет True, если дисциплина была удалена (вместе со всеми связями в таблице DisciplineSpecialty), иначе вернет False.

## 4. Получить дисциплину по ID

Информация, возвращаемая в случае удачного поиска дисциплины по ID:

| Параметр | Тип |
|---|---|
| id | Целое |
| name | Строка |
| code | Строка |
| specialities | Массив объектов (каждый объект содержит id, name, code) |

## 5. Получить список дисциплин по заданным параметрам

Информация, требуемая для получения списка дисциплин:

| Параметр | Тип | Описание |
|---|---|---|
| name | Строка | Фильтр по названию (поиск по подстроке, регистронезависимый) |
| code | Строка | Фильтр по коду (поиск по подстроке, регистронезависимый) |
| speciality_id | Целое | Фильтр по ID специальности (возвращает все дисциплины, связанные с этой специальностью) |

Информация возвращается в виде списка дисциплин. Каждый элемент списка имеет следующие параметры:

| Параметр | Тип |
|---|---|
| id | Целое |
| name | Строка |
| code | Строка |
| specialities | Массив объектов (каждый объект содержит id, name, code) |

## 6. Работа со специальностями

### 6.1 Получить все специальности

Возвращает список всех специальностей.

| Параметр | Тип |
|---|---|
| id | Целое |
| name | Строка |
| code | Строка |

### 6.2 Добавить специальность

Информация, требуемая для создания специальности:

| Параметр | Обязательность | Тип | Ограничение |
|---|---|---|---|
| name | Обязательно | Строка | не пустое, уникальное |
| code | Обязательно | Строка | не пустой, уникальный |

### 6.3 Получить специальность по ID

Возвращает информацию о специальности.

| Параметр | Тип |
|---|---|
| id | Целое |
| name | Строка |
| code | Строка |
| disciplines | Массив объектов (все дисциплины, связанные с этой специальностью) |

### 6.4 Изменить специальность по ID

Позволяет изменить name и/или code специальности.

### 6.5 Удалить специальность по ID

Удаляет специальность и все связи в таблице DisciplineSpecialty.
