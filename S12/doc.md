## ER-диаграмма и список связей

### Список реляционных связей:
*   `groups.id` → `curriculum.group_id`
*   `disciplines.id` → `curriculum.discipline_id`
*   `semesters.id` → `curriculum.semester_id`

        int id PK
        int semester_number
        string academic_year
        bool is_active
    }
