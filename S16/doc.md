### Вариант №16. Сервис корпусов (Campus)

#### Добавление Campus

Информация требуемая для создания Campus представлена в виде таблицы:

| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|----------|----------------|-----|-------------|----------------------|
| name | Да | string | Уникальное, 3-50 символов | — |
| address | Да | string | Не пустое | — |
| floors | Да | integer | Больше 0 | — |

Уникальные комбинации параметров: Отсутствуют. Идентификатор id генерируется автоматически.

Информация возвращаемая в случае удачного создания Campus:

| Параметр | Тип |
|----------|-----|
| id | integer |
| name | string |
| address | string |
| floors | integer |
| is_active | boolean |

---

#### Изменение Campus по ID

Информация требуемая для изменения Campus по ID представлена в виде таблицы:

| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|----------|----------------|-----|-------------|----------------------|
| name | Нет | string | Уникальное, 3-50 символов | — |
| address | Нет | string | Не пустое | — |
| floors | Нет | integer | Больше 0 | — |

Информация возвращаемая в случае удачного изменения Campus:

| Параметр | Тип |
|----------|-----|
| id | integer |
| name | string |
| address | string |
| floors | integer |
| is_active | boolean |

---

#### Удаление Campus по ID

Вернет True, если Campus была закрыта (удалена), иначе вернет False

---

Информация возвращаемая в случае удачного поиска Campus по ID:

| Параметр | Тип |
|----------|-----|
| id | integer |
| name | string |
| address | string |

---

#### Получить список Campus по заданным параметрам

Информация требуемая для получения списка Campus:

| Параметр | Тип | Описание |
|----------|-----|----------|
| min_floors | integer | Фильтр по минимальному количеству этажей |
| max_floors | integer | Фильтр по максимальному количеству этажей |
| exact_floors | integer | Точное количество этажей |
| address_contains | string | Поиск по частичному совпадению адреса |

Информация возвращается в виде списка Campus:

| Параметр | Тип |
|----------|-----|
| id | integer |
| name | string |
| address | string |
| floors | integer |
| is_active | boolean |
