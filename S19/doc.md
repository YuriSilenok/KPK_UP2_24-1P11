# Resource Pool Service (Вариант 19)

**Информация для создания**:

| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
|----------------|-----------|----------------|-----|-------------|----------------------|
| name | Название ресурса | Да | string | max 200 | - |
| type | Тип ресурса | Да | string | max 100 | - |
| quantity | Количество единиц | Да | int | >= 1 | 1 |
| location | Место хранения | Нет | string | max 255 | NULL |

**Уникальные комбинации параметров**: нет

**Информация при успешном создании**:

| Параметр | Тип |
|----------------|-----|
| id | int |
| name | string |
| type | string |
| quantity | int |
| location | string |
| is_active | bool |

#### 1.2 Изменить ресурс по ID

**Информация для изменения**:

| Параметр | Пояснение | Обязательность | Тип | Ограничение |
|----------------|-----------|----------------|-----|-------------|
| name | Название ресурса | Нет | string | max 200 |
| type | Тип ресурса | Нет | string | max 100 |
| quantity | Количество единиц | Нет | int | >= 1 |
| location | Место хранения | Нет | string | max 255 |

**Информация при успешном изменении**:

| Параметр | Тип |
|----------------|-----|
| id | int |
| name | string |
| type | string |
| quantity | int |
| location | string |
| is_active | bool |

#### 1.3 Удалить ресурс по ID (мягкое удаление)

**Возвращаемое значение**: `true` (если запись найдена и помечена удалённой), иначе `false`

#### 1.4 Получить ресурс по ID

**Информация при успешном получении**:

| Параметр | Пояснение | Тип |
|----------------|-----------|-----|
| id | Идентификатор | int |
| name | Название ресурса | string |
| type | Тип ресурса | string |
| quantity | Количество | int |
| location | Место хранения | string |
| is_active | Активен | bool |

#### 1.5 Получить список ресурсов

**Параметры запроса**:

| Параметр | Пояснение | Тип |
|----------------|-----------|-----|
| type | Фильтр по типу ресурса | string |
| is_active | Фильтр по активности | bool |
| limit | Лимит записей | int |
| offset | Смещение для пагинации | int |

**Возвращаемый список** (каждый элемент):

| Параметр | Тип |
|----------------|-----|
| id | int |
| name | string |
| type | string |
| quantity | int |
| location | string |
| is_active | bool |
