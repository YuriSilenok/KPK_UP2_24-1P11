# Вариант №2. Сервис профилей (Profile Service)

## Создание профиля

**Точка входа:**  
**Метод:** `POST` 
**URL:** `/profiles`

### Параметры тела запроса

| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|----------|----------------|-----|-------------|----------------------|
| first_name | Обязательно | Строка | Не пустое | — |
| last_name | Обязательно | Строка | Не пустое | — |
| middle_name | Не обязательно | Строка | — | — |
| phone | Обязательно | Строка | Уникальный, формат телефона | — |
| email | Обязательно | Строка | Уникальный, формат email | — |
| photo_url | Не обязательно | Строка | URL | — |

### Выходные данные

| Параметр | Тип |
|----------|-----|
| id | Целое |
| first_name | Строка |
| last_name | Строка |
| middle_name | Строка |
| phone | Строка |
| email | Строка |
| photo_url | Строка |

## Удаление профиля по ID

**Точка входа:**  
**Метод:** `DELETE` 
**URL:** `/profiles/{profile_id}`

### Параметры пути

| Параметр | Обязательность | Тип | Описание |
|----------|----------------|-----|----------|
| profile_id | Обязательно | Целое | Идентификатор профиля для поиска, обновления или удаления |

### Выходные данные

Вернет `{"deleted": true, "profile_id": id}`, если профиль был удален, иначе `404 Not Found`
