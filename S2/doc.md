# Вариант №2. Сервис профилей (Profile Service)

## Создание профиля

**Точка входа:**  
**Метод:** `POST`  
**URL:** `/profiles`

Информация требуемая для создания профиля пользователя

| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|----------|----------------|-----|-------------|----------------------|
| first_name | Обязательно | Строка | Не пустое | — |
| last_name | Обязательно | Строка | Не пустое | — |
| middle_name | Не обязательно | Строка | — | — |
| phone | Обязательно | Строка | Уникальный, формат телефона | — |
| email | Обязательно | Строка | Уникальный, формат email | — |
| photo_url | Не обязательно | Строка | URL | — |

Каждый параметр должен быть уникальный: phone и email.

Выходные данные

| Параметр | Тип |
|----------|-----|
| id | Целое |
| first_name | Строка |
| last_name | Строка |
| middle_name | Строка |
| phone | Строка |
| email | Строка |
| photo_url | Строка |

## Удаление профиля по ID

**Точка входа:**  
**Метод:** `DELETE`  
**URL:** `/profiles/{profile_id}`

Вернет True, если профиль был удален, иначе вернет False

## REST API 

### Базовый URL
http://localhost:8000

## Методы HTTP, используемые в API

| Метод | Назначение |
|-------|------------|
| POST | Создание нового ресурса (профиля) |
| GET | Получение данных (профиля или списка) |
| PUT | Полное обновление существующего ресурса |
| DELETE | Удаление ресурса |
| PATCH | Частичное обновление ресурса (настроек) |

## Параметры и их назначения для REST API

### Параметры пути (path)

| Параметр | Назначение |
|----------|------------|
| profile_id | Идентификатор профиля для поиска, обновления или удаления |

### Параметры запроса (query)

| Параметр | Назначение |
|----------|------------|
| first_name | Фильтрация списка профилей по имени |
| last_name | Фильтрация списка профилей по фамилии |
| phone | Фильтрация списка профилей по телефону |
| email | Фильтрация списка профилей по email |

### Параметры тела запроса (body) для POST /profiles и PUT /profiles/{profile_id}

| Параметр | Назначение |
|----------|------------|
| first_name | Имя пользователя |
| last_name | Фамилия пользователя |
| middle_name | Отчество пользователя (необязательно) |
| phone | Номер телефона пользователя (уникальный) |
| email | Адрес электронной почты пользователя (уникальный) |
| photo_url | Ссылка на фото профиля (необязательно) |

### Параметры тела запроса (body) для PATCH /profiles/{profile_id}/notifications

| Параметр | Назначение |
|----------|------------|
| phone_notification | Включение/отключение уведомлений по телефону |
| email_notification | Включение/отключение уведомлений по email |
