### Вариант №21. Сервис каникул и праздников (Holiday)

### Работа с праздниками (Holiday)

#### Добавление Holiday

Информация требуемая для создания Holiday

| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию | Пояснение |
|----------|----------------|-----|-------------|----------------------|-----------|
| name | Да | string | Уникальное, не пустое, до 100 символов | — | Название праздника |
| date | Да | date | Конкретная дата | — | Дата праздника |
| type | Да | integer | Ссылка на HolidayType | — | Тип (holiday или vacation) |

Информация возвращаемая в случае удачного создания Holiday

| Параметр | Тип | Пояснение |
|----------|-----|-----------|
| id | integer | Идентификатор |
| name | string | Название |
| date | date | Дата |
| type | integer | Тип |
| is_active | boolean | Статус активности |

Уникальные комбинации параметров: Отсутствуют. Идентификатор id генерируется автоматически.
