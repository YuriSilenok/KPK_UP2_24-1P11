### Вариант №21. Сервис каникул и праздников (Holiday)

### Работа с праздниками (Holiday)

#### Добавление Holiday

Информация требуемая для создания Holiday

| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|----------|----------------|-----|-------------|----------------------|
| name | Да | string | Уникальное, не пустое, до 100 символов | — |
| date | Да | date | Конкретная дата | — |
| type | Да | ForeignKey | Ссылка на HolidayType | — |

Информация возвращаемая в случае удачного создания Holiday

| Параметр | Тип |
|----------|-----|
| id | Целое |
| name | Строка |
| date | Дата |
| type | ForeignKey |
| is_active | Булево |

Уникальные комбинации параметров: Отсутствуют. Идентификатор id генерируется автоматически.

#### Изменение Holiday по ID

Информация требуемая для изменения Holiday по ID

| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|----------|----------------|-----|-------------|-----------------------|
| id | Да | Целое | Существует в БД | — |
| name | Нет | Строка | Уникальное, не пустое, до 100 символов | — |
| date | Нет | Дата | Не пустое | — |
| type | Нет | ForeignKey | Ссылка на HolidayType | — |
| is_active | Нет | Булево | — | — |

Информация возвращаемая в случае удачного изменения Holiday

| Параметр | Тип |
|----------|-----|
| id | Целое |
| name | Строка |
| date | Дата |
| type | ForeignKey |
| is_active | Булево |

#### Удаление Holiday по ID

Вернет True, если Holiday была закрыта (удалена), иначе вернет False

#### Получить Holiday по ID

Информация возвращаемая в случае удачного поиска Holiday по ID

| Параметр | Тип |
|----------|-----|
| id | Целое |
| name | Строка |
| date | Дата |
| type | ForeignKey |
| is_active | Булево |

#### Получить список Holiday по заданным параметрам

Информация требуемая для получения списка Holiday

| Параметр | Тип | Описание |
|----------|-----|----------|
| date_from | Дата | Фильтр по дате начала периода |
| date_to | Дата | Фильтр по дате окончания периода |
| type | ForeignKey | Фильтр по типу (ссылка на HolidayType) |

Информация возвращается в виде списка Holiday

| Параметр | Тип |
|----------|-----|
| id | Целое |
| name | Строка |
| date | Дата |
| type | ForeignKey |
| is_active | Булево |

### Работа с периодами каникул (VacationPeriod)

#### Добавление VacationPeriod

| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|----------|----------------|-----|-------------|----------------------|
| name | Да | string | Не пустое, до 100 символов | — |
| start_date | Да | date | Дата начала | — |
| end_date | Да | date | Дата окончания >= start_date | — |
| type | Да | ForeignKey | Ссылка на HolidayType | — |

#### Получить список VacationPeriod

| Параметр | Тип |
|----------|-----|
| id | integer |
| name | string |
| start_date | date |
| end_date | date |
| type | ForeignKey |
| is_active | boolean |

Уникальные комбинации параметров: Отсутствуют. Идентификатор id генерируется автоматически.

### Работа с типами праздников (HolidayType)

#### Добавление HolidayType

| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|----------|----------------|-----|-------------|----------------------|
| name | Да | string | Уникальное, не пустое, до 50 символов | — |
| code | Да | string | Уникальное, "holiday" или "vacation" | — |

#### Получить список HolidayType

| Параметр | Тип |
