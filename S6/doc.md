# Отчет по сервису специальностей (Вариант 6)

## Функционал
- Учет специальностей (код, название).
- Учет стандартов ФГОС.
- Связь специальностей с ФГОС (многие-ко-многим).

## ER-диаграмма
![ER-диаграмма](erd.png)

## Параметры сущности "Специальность"


| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
| :--- | :--- | :--- | :--- | :--- |
| code | Да | String | Unique | - |
| name | Да | String | Not Null | - |

**Уникальные комбинации:** `code`.

## Получить список специальностей

| Параметр | Тип | Описание |
| :--- | :--- | :--- |
| limit | Integer | Кол-во записей |

**Возвращаемый список:**

| Параметр | Тип |
| :--- | :--- |
| id | Integer |
| code | String |
| name | String |
