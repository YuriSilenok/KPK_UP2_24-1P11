# Вариант №7. Сервис групп (Group Service)

---

## Сущность: Учебная группа (groups)

### Перечень уникальных комбинаций и ограничений параметров
Данные ограничения целостности и бизнес-правила заложены в архитектуру сервиса и проверяются на уровне СУБД:
1. **Уникальность наименования:** Поле `name` является строго уникальным (`UNIQUE`). Наличие двух учебных групп с абсолютно одинаковым названием в базе данных запрещено.
2. **Ограничение года формирования:** Поле `formation_year` содержит логический констрейнт (`CHECK >= 2000`). Год создания учебной группы не может быть хронологически меньше 2000 года.
3. **Ограничение базы обучения:** Поле `education_base` может принимать только два фиксированных значения: `9` или `11` (`CHECK IN (9, 11)`), что соответствует базе 9-го или 11-го класса школы. Другие значения СУБД не примет.
4. **Обязательность заполнения (NOT NULL):** Поля `name`, `formation_year`, `education_base` и `curator_id` являются строго обязательными для заполнения при создании записи. Поле `is_active` заполняется автоматически значением по умолчанию, если не передано явно.

---

### Добавить группу
Информация, требуемая для создания группы колледжа.

**Входные параметры**


| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
| :--- | :--- | :--- | :--- | :--- |
| name | Обязательно | Строка | Уникальное (`UNIQUE`) | — |
| formation_year | Обязательно | Целое | `>= 2000` | — |
| education_base | Обязательно | Целое | `9` или `11` | — |
| is_active | Не обязательно | Логический | — | `True` |
| curator_id | Обязательно | Целое | Внешний ID | — |

**Выходные данные**


| Параметр | Тип | Описание |
| :--- | :--- | :--- |
| id | Целое | Уникальный ID созданной группы |
| name | Строка | Название группы |
| formation_year | Целое | Год формирования |
| education_base | Целое | База образования (9 или 11 класс) |
| is_active | Логический | Статус группы (True - активна, False - выпустилась) |
| curator_id | Целое | Внешний ID куратора |
| message | Строка | Текст результата: "Группа успешно создана" |

### Изменить группу по ID
Входные параметры для обновления данных группы.

**Входные параметры**


| Параметр | Обязательность | Тип | Ограничение |
| :--- | :--- | :--- | :--- |
| id | Обязательно | Целое | Существует в БД |
| education_base | Нет | Целое | `9` или `11` |
| is_active | Нет | Логический | Изменение статуса группы |
| curator_id | Нет | Целое | Внешний ID |

**Выходные данные**


| Параметр | Тип | Описание |
| :--- | :--- | :--- |
| id | Целое | ID измененной записи |
| result | Логический | True, если запись успешно изменена |
