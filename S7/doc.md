# Вариант №7. Сервис групп (Group Service)

---

## 1. Сущность: Статус группы (statusgroup)

### Добавить статус
Информация требуемая для создания нового статуса

**Входные параметры**


| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
| :--- | :--- | :--- | :--- | :--- |
| title | Обязательно | Строка | Уникальное, не пустое | — |

**Выходные данные**


| Параметр | Тип | Описание |
| :--- | :--- | :--- |
| id | Целое | Уникальный идентификатор созданного статуса |
| title | Строка | Название созданного статуса |

---

## 2. Сущность: Учебная группа (groups)

### Добавить группу
Информация требуемая для создания группы колледжа

**Входные параметры**


| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
| :--- | :--- | :--- | :--- | :--- |
| name | Обязательно | Строка | Уникальное | — |
| formation_year | Обязательно | Целое | >= 2000 | — |
| education_base | Обязательно | Целое | 9 или 11 | — |
| status_id | Не обезательно| Целое | FK к statusgroup | 1 |
| curator_id | Обязательно | Целое | Внешний ID | — |

**Выходные данные**


| Параметр | Тип | Описание |
| :--- | :--- | :--- |
| id | Целое | Уникальный ID созданной группы |
| name | Строка | Название группы |
| formation_year | Целое | Год формирования |
| education_base | Целое | База образования (9 или 11 класс) |
| status_id | Целое | ID текущего статуса |
| curator_id | Целое | Внешний ID куратора |
| message | Строка | Текст результата: "Группа успешно создана" |

### Изменить группу по ID
Входные параметры для обновления данных группы

**Входные параметры**


| Параметр | Обязательность | Тип | Ограничение |
| :--- | :--- | :--- | :--- |
| education_base | Нет | Целое | 9 или 11 |
| status_id | Нет | Целое | FK к statusgroup |
| curator_id | Нет | Целое | Внешний ID |

**Выходные данные**


| Параметр | Тип | Описание |
| :--- | :--- | :--- |
| id | Целое | ID измененной записи |
| result | Логический | True, если запись успешно изменена |

### Получить список групп
Входные параметры для фильтрации данных

**Входные параметры**


| Параметр | Обязательность | Тип | Описание |
| :--- | :--- | :--- | :--- |
| formation_year | Нет | Целое | Фильтрация списка по году набора |
| education_base | Нет | Целое | Фильтрация по базе обучения (9 или 11) |

**Выходные данные (Список объектов)**


| Параметр | Тип | Описание |
| :--- | :--- | :--- |
| id | Целое | ID записи в БД |
| name | Строка | Название учебной группы |
| formation_year | Целое | Год набора |
| education_base | Целое | База образования |
| status_id | Целое | Текущий статус группы |
| curator_id | Целое | Идентификатор куратора |

### Удаление по ID
Удаление записи группы из базы данных

**Входные параметры**


| Параметр | Обязательность | Тип | Ограничение |
| :--- | :--- | :--- | :--- |
| id | Обязательно | Целое | Существует в БД |

**Выходные данные**


| Параметр | Тип | Описание |
| :--- | :--- | :--- |
| result | Логический | True, если запись удалена успешно |
