## 1. Сущность: Статус группы (statusgroup)
Справочник состояний учебной группы (например: Активна, Выпустилась).

### Добавить статус
Информация требуемая для создания нового статуса


| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
| :--- | :--- | :--- | :--- | :--- |
| title | Обязательно | Строка | Уникальное, не пустое | — |

**Выходные данные**


| Параметр | Тип |
| :--- | :--- |
| id | Целое |
| title | Строка |

### Изменить статус по ID
Входные параметры


| Параметр | Обязательность | Тип | Ограничение |
| :--- | :--- | :--- | :--- |
| title | Обязательно | Строка | Уникальное |

**Выходные данные**


| Параметр | Тип | Описание |
| :--- | :--- | :--- |
| result | Логический | True, если статус изменен |

### Удалить статус по ID
Вернет **True**, если запись удалена, иначе **False**.

### Получить список статусов
**Выходные данные (Список объектов)**


| Параметр | Тип | Описание |
| :--- | :--- | :--- |
| id | Целое | ID статуса |
| title | Строка | Название статуса |

## 2. Сущность: Учебная группа (groups)
Основная сущность для учета групп колледжа.

### Добавить группу
Информация требуемая для создания группы


| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
| :--- | :--- | :--- | :--- | :--- |
| name | Обязательно | Строка | Уникальное | — |
| formation_year | Обязательно | Целое | >= 2000 | — |
| education_base | Обязательно | Целое | 9 или 11 | — |
| status_id | Обязательно | Целое | FK к statusgroup | 1 |
| curator_id | Обязательно | Целое | Внешний ID | — |

**Выходные данные**


| Параметр | Тип | Описание |
| :--- | :--- | :--- |
| message | Строка | Текст результата: "Группа успешно создана" |

### Изменить группу по ID
Входные параметры


| Параметр | Обязательность | Тип | Ограничение |
| :--- | :--- | :--- | :--- |
| education_base | Нет | Целое | 9 или 11 |
| status_id | Нет | Целое | FK к statusgroup |
| curator_id | Нет | Целое | Внешний ID |

**Выходные данные**


| Параметр | Тип | Описание |
| :--- | :--- | :--- |
| result | Логический | True, если запись успешно изменена |

### Получить группу по ID (Просмотр)
**Выходные данные**


| Параметр | Тип |
| :--- | :--- |
| id | Целое |
| name | Строка |
| formation_year | Целое |
| education_base | Целое |
| status_id | Целое |
| curator_id | Целое |

### Удалить группу по ID
Вернет **True**, если запись удалена.
