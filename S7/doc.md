## 1. Сущность: Статус группы (statusgroup)

### Добавить статус
Информация требуемая для создания нового статуса


| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
| :--- | :--- | :--- | :--- | :--- |
| title | Обязательно | Строка | Уникальное, не пустое | — |

**Выходные данные**


| Параметр | Тип | Описание |
| :--- | :--- | :--- |
| id | Целое | Уникальный ID созданного статуса |
| title | Строка | Название созданного статуса |

### Изменить статус по ID
**Выходные данные**


| Параметр | Тип | Описание |
| :--- | :--- | :--- |
| id | Целое | ID измененного статуса |
| title | Строка | Новое название статуса |
| result | Логический | True, если запись успешно изменена |

### Удалить статус по ID
**Выходные данные**


| Параметр | Тип | Описание |
| :--- | :--- | :--- |
| result | Логический | True, если запись удалена |

## 2. Сущность: Учебная группа (groups)

### Добавить группу
Информация требуемая для создания группы


| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
| :--- | :--- | :--- | :--- | :--- |
| name | Обязательно | Строка | Уникальное | — |
| formation_year | Обязательно | Целое | >= 2000 | — |
| education_base | Обязательно | Целое | 9 или 11 | — |
| status_id | Обязательно | Целое | FK к statusgroup | 1 |
| curator_id | Обязательно | Целое | Внешний ID | — |

**Выходные данные**


| Параметр | Тип | Описание |
| :--- | :--- | :--- |
| id | Целое | Уникальный ID созданной группы |
| name | Строка | Название учебной группы |
| formation_year | Целое | Год формирования |
| education_base | Целое | База образования (9/11) |
| status_id | Целое | ID установленного статуса |
| curator_id | Целое | Внешний ID куратора |
| message | Строка | Текст результата: "Группа успешно создана" |

### Изменить группу по ID
**Выходные данные**


| Параметр | Тип | Описание |
| :--- | :--- | :--- |
| id | Целое | ID измененной группы |
| education_base | Целое | Новое значение базы |
| status_id | Целое | Новый ID статуса |
| curator_id | Целое | Новый ID куратора |
| result | Логический | True, если запись успешно изменена |

### Получить группу по ID
**Выходные данные**


| Параметр | Тип | Описание |
| :--- | :--- | :--- |
| id | Целое | ID записи в БД |
| name | Строка | Название группы |
| formation_year | Целое | Год формирования |
| education_base | Целое | База (9/11) |
| status_id | Целое | ID статуса |
| curator_id | Целое | ID куратора |

### Удалить группу по ID
**Выходные данные**


| Параметр | Тип | Описание |
| :--- | :--- | :--- |
| result | Логический | True, если группа удалена |
