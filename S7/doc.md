# Вариант №7. Сервис групп (Group Service)
---

## Сущность: Учебная группа (groups)

### Перечень уникальных комбинаций и ограничений параметров
1. **Составная уникальность группы (Название + Год):** Поля `name` и `formation_year` образуют составной уникальный индекс. Названия групп могут циклично повторяться в разные годы набора, но в рамках одного и того же года дублирование названий запрещено.
2. **Ограничение года формирования:** Поле `formation_year` имеет ограничение (`CHECK >= 2000`). Год создания группы не может быть меньше 2000.
3. **Ограничение базы обучения:** Поле `education_base` принимает строго фиксированные значения: `9` или `11` (`CHECK IN (9, 11)`).
4. **Обязательность заполнения:** Поля `name`, `formation_year` и `education_base` обязательны при создании записи.
5. **Статус активности при создании:** Параметр `is_active` удален из входных данных создания. Новая группа всегда автоматически создаётся со статусом `True`.
6. **Необязательность куратора:** Поле `curator_id` может оставаться пустым (`NULL`), если куратор ещё не назначен.

### Добавить группу
Информация, требуемая для создания группы колледжа.

**Входные параметры**


| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
| :--- | :--- | :--- | :--- | :--- |
| name | Обязательно | Строка | Уникально в рамках года | — |
| formation_year | Обязательно | Целое | `>= 2000` | — |
| education_base | Обязательно | Целое | `9` или `11` | — |
| curator_id | Не обязательно | Целое | Внешний ID | `NULL` |

**Выходные данные**


| Параметр | Тип | Описание |
| :--- | :--- | :--- |
| id | Целое | Уникальный ID созданной группы |
| name | Строка | Название группы |
| formation_year | Целое | Год формирования |
| education_base | Целое | База образования (9 или 11 класс) |
| is_active | Логический | Статус группы (Всегда True при создании) |
| curator_id | Целое / NULL | Внешний ID куратора |
| message | Строка | Текст результата: "Группа успешно создана" |

### Изменить группу по ID
Входные параметры для обновления данных группы.

**Входные параметры**


| Параметр | Обязательность | Тип | Ограничение |
| :--- | :--- | :--- | :--- |
| id | Обязательно | Целое | Существует в БД |
| education_base | Нет | Целое | `9` или `11` |
| is_active | Нет | Логический | Изменение статуса группы |
| curator_id | Нет | Целое | Внешний ID / NULL |

**Выходные данные**


| Параметр | Тип | Описание |
| :--- | :--- | :--- |
| id | Целое | ID измененной записи |
| result | Логический | True, если запись успешно изменена |
