# Вариант №7. Сервис групп (Group Service)

---

## Сущность: Учебная группа (groups)

### Перечень уникальных комбинаций и ограничений параметров
Данные ограничения целостности и бизнес-правила заложены в архитектуру сервиса и проверяются на уровне СУБД:
1. **Составной уникальный ключ:** Поля `name` и `formation_year` образуют единый составной уникальный ключ `[name + formation_year]`. Названия групп могут циклично повторяться в разные годы набора, но в рамках одного и того же года дублирование названий строго запрещено.
2. **Ограничение года формирования:** Поле `formation_year` содержит логический констрейнт (`CHECK >= 2000`). Год создания учебной группы не может быть хронологически меньше 2000 года.
3. **Ограничение базы обучения:** Поле `education_base` может принимать только два фиксированных значения: `9` или `11` (`CHECK IN (9, 11)`), что соответствует базе 9-го или 11-го класса школы.
4. **Обязательность заполнения (NOT NULL):** Поля `name`, `formation_year` и `education_base` являются строго обязательными для заполнения при создании записи.
5. **Статус активности при создании:** Параметр `is_active` запрещено передавать на вход при создании. База данных автоматически инициализирует новую группу со статусом `True`.
6. **Необязательность куратора:** Поле `curator_id` может оставаться пустым (`NULL`), если на момент формирования группы куратор ещё не назначен.

### Добавить группу
Информация, требуемая для создания группы колледжа.

**Входные параметры**



| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
| :--- | :--- | :--- | :--- | :--- |
| name | Обязательно | Строка | Составной уникальный ключ `[name + formation_year]` | — |
| formation_year | Обязательно | Целое | `>= 2000`, входит в составной уникальный ключ | — |
| education_base | Обязательно | Целое | `9` или `11` | — |
| curator_id | Не обязательно | Целое | Внешний ID | `NULL` |

**Выходные данные**



| Параметр | Тип | Описание |
| :--- | :--- | :--- |
| id | Целое | Уникальный ID созданной группы (автоинкремент) |
| name | Строка | Название группы |
| formation_year | Целое | Год формирования |
| education_base | Целое | База образования (9 или 11 класс) |
| is_active | Логический | Статус группы (Всегда True при создании) |
| curator_id | Целое / NULL | Внешний ID куратора (если назначен) |
| message | Строка | Текст результата: "Группа успешно создана" |

### Изменить группу по ID
Входные параметры для обновления данных группы.

**Входные параметры**



| Параметр | Обязательность | Тип | Ограничение |
| :--- | :--- | :--- | :--- |
| id | Обязательно | Целое | Существует в БД |
| education_base | Нет | Целое | `9` или `11` |
| is_active | Нет | Логический | Изменение статуса группы |
| curator_id | Нет | Целое | Внешний ID / NULL |

**Выходные данные**



| Параметр | Тип | Описание |
| :--- | :--- | :--- |
| id | Целое | ID измененной записи |
| result | Логический | True, если запись успешно изменена |
