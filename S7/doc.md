# Вариант №7. Сервис групп (Group Service)
**Исполнитель:** Столяров Артем Сергеевич
**Группа:** 24-1П11

## Функционал
* Регистрация и учет учебных групп колледжа.
* Учет года формирования и образовательной базы (9 или 11 класс).
* Управление статусом группы (Активна/Выпустилась).
* Закрепление куратора через внешний идентификатор (ID).

---

## Добавить статус в таблицу statusgroup
Информация требуемая для создания нового статуса


| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
| :--- | :--- | :--- | :--- | :--- |
| title | Да | Строка | Уникальное, не пустое | — |

**Информация возвращаемая при успехе:**


| Параметр | Тип |
| :--- | :--- |
| id | Целое |
| title | Строка |

---

## Добавить группу
Информация требуемая для создания группы


| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
| :--- | :--- | :--- | :--- | :--- |
| name | Да | Строка | Уникальное | — |
| formation_year | Да | Целое | >= 2000 | — |
| education_base | Да | Целое | 9 или 11 | — |
| status_id | Да | Целое | FK | 1 |
| curator_id | Да | Целое | Внешний ID | — |

**Выходные данные при успехе:**
* message: "Группа успешно создана"

---

## Изменить группу по ID

| Параметр | Обязательность | Тип |
| :--- | :--- | :--- |
| education_base | Нет | Целое |
| status_id | Нет | Целое |
| curator_id | Нет | Целое |

---

## Удаление по ID
Вернет **True**, если запись удалена.
