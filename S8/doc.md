# Вариант №8. Сервис подгрупп (Subgroup Service)

## Создать подгруппу

Информация, требуемая для создания подгруппы:

| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|----------|---------------|-----|-------------|----------------------|
| name | Обязательно | Строка | Не более 100 символов | — |
| type | Обязательно | Строка | `language`, `sport`, `other` | — |
| group_id | Обязательно | Целое | Внешний ключ к группе | — |

**Уникальная комбинация:** `(group_id, name)`

### Выходные данные

| Параметр | Тип |
|----------|-----|
| id | Целое |
| name | Строка |
| type | Строка |
| group_id | Целое |

---

## Изменить подгруппу по ID

| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|----------|---------------|-----|-------------|----------------------|
| name | Необязательно | Строка | Не более 100 символов | Не меняется |
| type | Необязательно | Строка | `language`, `sport`, `other` | Не меняется |
| group_id | Необязательно | Целое | Внешний ключ к группе | Не меняется |

### Выходные данные

| Параметр | Тип |
|----------|-----|
| id | Целое |
| name | Строка |
| type | Строка |
| group_id | Целое |

---

## Удалить подгруппу по ID

Вернёт `True`, если подгруппа была удалена, иначе `False`
