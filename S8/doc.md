# Вариант №8. Сервис подгрупп (Subgroup Service)

## 1. Создать подгруппу
**POST** `/subgroups`

**Информация для создания:**

| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|----------|---------------|-----|-------------|----------------------|
| type | Обязательно | Строка | `language`, `sport`, `other` | — |
| group_id | Обязательно | Целое | Внешний ключ к группе | — |

**Уникальная комбинация:** `(group_id, type)`

**Выходные данные:**

| Параметр | Тип |
|----------|-----|
| id | Целое |
| type | Строка |
| group_id | Целое |

---

## 2. Изменить подгруппу по ID
**PATCH** `/subgroups/{id}`

**Информация для изменения:**

| Параметр | Обязательность | Тип | Ограничение | Значение по умолчанию |
|----------|---------------|-----|-------------|----------------------|
| type | Необязательно | Строка | `language`, `sport`, `other` | — |

**Примечание:** Если параметр `type` не передан в запросе, то тип подгруппы остаётся без изменений.

**Выходные данные:**

| Параметр | Тип |
|----------|-----|
| id | Целое |
| type | Строка |
| group_id | Целое |

---

## 3. Удалить подгруппу по ID
**DELETE** `/subgroups/{id}`

**Возвращает:** `True`, если подгруппа была удалена, иначе `False`

---

## 4. Получить подгруппу по ID
**GET** `/subgroups/{id}`

**Выходные данные:**

| Параметр | Тип |
|----------|-----|
| id | Целое |
| type | Строка |
| group_id | Целое |

---

## 5. Получить список подгрупп по заданным параметрам
**GET** `/subgroups`

**Входные параметры (query params):**

| Параметр | Тип | Обязательность | Описание |
|----------|-----|---------------|----------|
| group_id | Целое | Необязательно | Фильтр по группе. Если не указан — фильтрация не применяется |
| type | Строка | Необязательно | Фильтр по типу (`language`, `sport`, `other`). Если не указан — фильтрация не применяется |

**Выходные данные (массив объектов):**

| Параметр | Тип |
|----------|-----|
| id | Целое |
| type | Строка |
| group_id | Целое |
