# Вариант №8. Сервис подгрупп (Subgroup Service)

## Модель данных (ER-диаграмма)

| Таблица | Поля |
|---------|------|
| subgroups | id (PK), group_id (FK), name, discipline_id (FK), is_active |
| subgroup_students | id (PK), subgroup_id (FK), student_id (FK) |

---

## 1. Создать подгруппу

**POST** `/subgroups`

**Вход:**

| Параметр | Обязательность | Тип | Ограничение |
|----------|---------------|-----|-------------|
| name | Обязательно | Строка | - |
| group_id | Обязательно | Целое | Внешний ключ к группе |
| discipline_id | Обязательно | Целое | Внешний ключ к дисциплине |

**Уникальная комбинация:** `(group_id, name)`

**Выход:**

| Параметр | Тип |
|----------|-----|
| id | Целое |
| name | Строка |
| group_id | Целое |
| discipline_id | Целое |
| is_active | Булевый |

---

## 2. Изменить подгруппу по ID

**PATCH** `/subgroups/{id}`

**Вход:**

| Параметр | Обязательность | Тип | Ограничение |
|----------|---------------|-----|-------------|
| name | Опционально | Строка | - |
| group_id | Опционально | Целое | Внешний ключ к группе |
| discipline_id | Опционально | Целое | Внешний ключ к дисциплине |

**Выход:**

| Параметр | Тип |
|----------|-----|
| id | Целое |
| name | Строка |
| group_id | Целое |
| discipline_id | Целое |
| is_active | Булевый |

---

## 3. Удалить подгруппу по ID

**DELETE** `/subgroups/{id}`

**Логическое удаление (soft delete):** устанавливается `is_active = False`

**Возвращает:**

| Параметр | Тип |
|----------|-----|
| deleted | Булевый |
| id | Целое |
| is_active | Булевый |

**Пример:** `{"deleted": True, "id": 1, "is_active": False}`

---

## 4. Получить подгруппу по ID

**GET** `/subgroups/{id}`

**Выход:**

| Параметр | Тип |
|----------|-----|
| id | Целое |
| name | Строка |
| group_id | Целое |
| discipline_id | Целое |
| is_active | Булевый |

---

## 5. Получить список подгрупп по заданным параметрам

**GET** `/subgroups`

**Вход (query params):**

| Параметр | Тип | Обязательность |
|----------|-----|---------------|
| group_id | Целое | Опционально |
| name | Строка | Опционально |
| discipline_id | Целое | Опционально |

**Выход (массив):**

| Параметр | Тип |
|----------|-----|
| id | Целое |
| name | Строка |
| group_id | Целое |
| discipline_id | Целое |
| is_active | Булевый |

**Важно:** возвращаются только подгруппы с `is_active = True`

---

## 6. Добавить студента в подгруппу

**POST** `/subgroup-students`

**Вход:**

| Параметр | Обязательность | Тип | Ограничение |
|----------|---------------|-----|-------------|
| subgroup_id | Обязательно | Целое | Внешний ключ к подгруппе |
| student_id | Обязательно | Целое | Внешний ключ к студенту |

**Уникальная комбинация:** `(subgroup_id, student_id)`

**Выход:**

| Параметр | Тип |
|----------|-----|
| id | Целое |
| subgroup_id | Целое |
| student_id | Целое |

---

## 7. Удалить студента из подгруппы

**DELETE** `/subgroup-students/{id}`

**Возвращает:**

| Параметр | Тип |
|----------|-----|
| deleted | Булевый |

**Пример:** `{"deleted": True}`

---

## 8. Получить всех студентов подгруппы

**GET** `/subgroups/{id}/students`

**Выход (массив):**

| Параметр | Тип |
|----------|-----|
| id | Целое |
| subgroup_id | Целое |
| student_id | Целое |
