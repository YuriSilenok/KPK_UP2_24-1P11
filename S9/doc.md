# Вариант 9. Student Movement Service (Сервис движения студентов):
## 1. Сущность «StudentMovement»

### 1.1. Добавить StudentMovement

**Информация, требуемая для создания StudentMovement:**

| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
|----------|-----------|----------------|-----|-------------|----------------------|
| student_id | ID студента (из внешнего сервиса) | Обязательно | Целое число | Должен существовать | - |
| movement_type_id | ID типа движения | Обязательно | Целое число | Должен существовать | - |
| from_group_id | ID группы, из которой переводится студент | Нет | Целое число | Может быть NULL | NULL |
| to_group_id | ID группы, в которую переводится студент | Нет | Целое число | Может быть NULL | NULL |
| movement_date | Дата движения | Обязательно | Дата | - | - |
| reason | Причина движения | Обязательно | Текст | от 1 до 500 символов | - |
| order_number | Номер приказа | Обязательно | Строка | Уникальный | - |
| created_by | Кто создал запись | Обязательно | Строка | от 1 до 100 символов | - |

**Уникальные комбинации параметров:**
- `order_number`

**Информация, возвращаемая в случае удачного создания StudentMovement:**

| Параметр | Тип |
|----------|-----|
| id | Целое число |
| student_id | Целое число |
| movement_type_id | Целое число |
| from_group_id | Целое число |
| to_group_id | Целое число |
| movement_date | Дата |
| reason | Строка |
| order_number | Строка |
| created_by | Строка |
| is_active | Булево значение |

---

### 1.2. Изменить StudentMovement по ID

**Информация, требуемая для изменения StudentMovement по ID:**

| Параметр | Пояснение | Обязательность | Тип | Ограничение |
|----------|-----------|----------------|-----|-------------|
| reason | Причина движения | Нет | Текст | от 1 до 500 символов |
| to_group_id | ID группы назначения | Нет | Целое число | Должен существовать |
| movement_date | Дата движения | Нет | Дата | - |

**Примечание:** поле `order_number` не может быть изменено.

**Информация, возвращаемая в случае удачного изменения StudentMovement:**

| Параметр | Тип |
|----------|-----|
| id | Целое число |
| student_id | Целое число |
| movement_type_id | Целое число |
| from_group_id | Целое число |
| to_group_id | Целое число |
| movement_date | Дата |
| reason | Строка |
| order_number | Строка |
| created_by | Строка |
| is_active | Булево значение |

---

### 1.3. Удалить StudentMovement по ID

Вернет `True`, если запись была помечена как удалённая, иначе `False`.

Фактически запись из БД не удаляется, а реализуется через булевое поле `is_active`.

При попытке повторного удаления уже неактивной записи сервис возвращает `False`.
