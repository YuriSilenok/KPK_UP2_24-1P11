# 9. Student Movement Service (Сервис движения студентов):
## 1. Сущность «Движение студента» (StudentMovement)

### 1.1. Создание движения студента

**Информация, требуемая для создания движения:**

| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
|----------|-----------|----------------|-----|-------------|----------------------|
| student_id | ID студента | Обязательно | Целое число | Должен существовать | - |
| movement_type_id | ID типа движения | Обязательно | Целое число | Должен существовать | - |
| from_group_id | ID группы, из которой переводится студент | Нет | Целое число | Может быть NULL | NULL |
| to_group_id | ID группы, в которую переводится студент | Нет | Целое число | Может быть NULL | NULL |
| movement_date | Дата движения | Обязательно | Дата | Не может быть в будущем | - |
| reason | Причина движения | Обязательно | Текст | от 1 до 500 символов | - |
| order_number | Номер приказа | Обязательно | Строка | Уникальный | - |
| created_by | Кто создал запись | Обязательно | Строка | от 1 до 100 символов | - |

**Уникальные комбинации параметров:**
- `order_number` — номер приказа должен быть уникальным

**Выходные данные при успешном создании:**

| Параметр | Тип |
|----------|-----|
| id | Целое число |
| student_id | Целое число |
| movement_type_id | Целое число |
| from_group_id | Целое число |
| to_group_id | Целое число |
| movement_date | Дата |
| reason | Строка |
| order_number | Строка |
| created_by | Строка |
| is_active | Булево значение |

### 1.3. Удалить движение студента по ID (мягкое удаление)

Вернет `True`, если запись была помечена как удалённая, иначе `False`.

Фактически запись из БД не удаляется, а реализуется через булевое поле `is_active`.

## 2. Справочные сущности (описание структуры)

### 2.1. Сущность «Студент» (Student)

| Поле | Тип | Ограничение |
|------|-----|-------------|
| id | Целое число | PRIMARY KEY |
| student_number | Строка | UNIQUE |
| current_group_id | Целое число | FOREIGN KEY → Groups |
| status | Строка | NOT NULL |

### 2.2. Сущность «Группа» (Group)

| Поле | Тип | Ограничение |
|------|-----|-------------|
| id | Целое число | PRIMARY KEY |
| speciality | Строка | NOT NULL |
| course_year | Целое число | от 1 до 4 |

### 2.3. Сущность «Тип движения» (MovementType)

| Поле | Тип | Ограничение |
|------|-----|-------------|
| id | Целое число | PRIMARY KEY |
| name | Строка | UNIQUE |

**Допустимые значения name:**
- перевод
- отчисление
- восстановление
- академический отпуск
